# Day 6 -- Object-Oriented Programming (OOP) Basics

## Overview
This project demonstrates basic Object-Oriented Programming (OOP) concepts in Python by building a `Student` class.

The implementation covers:

- Classes and objects
- `__init__` constructor
- Attributes
- Instance variables
- Methods using `self`
- `__str__` method
- Unit testing using `pytest`
